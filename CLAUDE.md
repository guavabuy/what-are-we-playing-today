# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 这个项目是什么

一部作品。形式就是作者和 Claude 的聊天记录，原样保留，不加工。
用幼儿园隐喻国际政治格局。

## 核心命题

历史上所有的战争，本质都是在争一件事：谁来定规则。
幼儿园里的孩子们，争的从来不是玩具，是谁决定今天玩什么游戏。

## 精神

不说教。他们会留下什么，就留下什么。

## 项目结构

```
今天玩什么/
├── 世界观.md        ← 核心设定（必读）
├── 角色/            ← 每个角色一个文件，YAML frontmatter 记录设定和状态
├── 剧本/            ← 按时间线分集，每集 = 一次聊天记录
├── 短片/            ← 独立成篇的精华片段（作者指定才写）
└── 循环？历史？.md   ← 最初的灵感
```

## 写作风格原则

- 日式场景：镜头平，水面下有东西，不加多余意象
- 美式场景：直接，口语，动作驱动
- 中式场景：话少，意在言外
- **不往里塞意象，除非作者要求**
- **Claude 不是作者，不要擅自写内容。作者说想法，Claude 帮忙落成剧本格式。**

## 每次聊天的工作流

1. 作者丢想法 → Claude 帮忙整理成剧本场景
2. 更新 `角色/*.md` 的 status 字段
3. 如有新设定，更新 `世界观.md`
4. commit + push 到 GitHub
5. 世界观和角色卡片保持同步

## 协作规则

- 修改角色文件时，保持 YAML frontmatter 格式一致
- 写新内容前先读 `世界观.md`
- 不要自行添加新角色或改变映射关系
- 隐喻让读者自己领会，不解释
